Publish a message

Every user has an append-only log of JSON messages, called their feed.

Use this command to publish your first message:

bash
js
sbot publish --type post --text "Hello, world!"
sbot.publish({
  type: 'post',
  text: 'Hello, world!'
}, function (err, msg) {
  // 'msg' includes the hash-id and headers
})
