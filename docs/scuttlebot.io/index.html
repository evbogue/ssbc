Scuttlebot
a peer-to-peer log store

Scuttlebot is an open source peer-to-peer log store used as a database, identity provider, and messaging system. It features global replication, file-syncronization, and end-to-end encryption.

Install the database

bash
npm install -g ssb-server

Install the database to your device, not to a host. Each user runs their own Scuttlebot.

Start the server

bash
ssb-server start
Scuttlebot server must be running for any of its other commands, or apps, to work.
