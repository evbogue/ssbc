Sinks

A Sink is a stream that is not readable. You must have a sink at the end of a pipeline for data to move towards.

You can only use one sink per pipeline.

js
pull(source, through, sink)

See also:

  • Sources
  • Throughs

drain (op?, done?)

Drain the stream, calling op on each data. call done when stream is finished. If op returns ===false, abort the stream.

reduce (reduce, initial, cb)

reduce stream into single value, then callback.

collect(cb)

Read the stream into an array, then callback.

onEnd (cb)

Drain the stream and then callback when done.

log

output the stream to console.log
