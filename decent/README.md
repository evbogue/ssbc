# Decent

Decent is a browser UI for Secure Scuttlebutt, served by `plugins/decent-ui.js`
on port `8888`.  It connects to the running sbot via WebSocket.

## Layout

```
decent/
├── src/                 # all frontend source
│   ├── main.js          # entry point; wires depject
│   ├── wire.js          # first / map / asyncConcat plug combinators
│   ├── config.js        # runtime config (remote address, etc.)
│   ├── keys.js          # identity keys (localStorage in browser)
│   ├── manifest.json    # muxrpc manifest for the sbot RPC surface
│   ├── style.css        # single stylesheet, no preprocessor
│   └── modules/
│       ├── core/        # sbot connection, screen routing, menu, blob URLs
│       ├── ui/          # posts, feed, avatars, compose, profile, etc.
│       ├── git/         # git-forge UI (repos, browser, search, activity)
│       └── extras/      # channels, notifications, search, key, etc.
├── scripts/             # style.js, postprocess-index.js
└── build/               # generated by `npm run build:web` — do not edit
```

Modules use a plugin pattern: each exports `{ needs, gives, create(api) }`.
`needs` declares what the module consumes from other modules, `gives` declares
what it provides, and depject wires them at startup.  `'first'` means the first
registered implementation wins; `'map'` means all implementations run and their
outputs are merged.

## Building

From the repo root:

```bash
npm install          # installs everything (one package.json, one lockfile)
npm run build:web    # rebuilds decent/build/
```

Output files:

- `decent/build/index.html`
- `decent/build/style.css`

## Running

Start the sbot server (`npm start` from repo root), then open:

```
http://127.0.0.1:8888/
```

The server injects the WebSocket remote address into the page automatically.

## Configuration

Override host/port in `~/.ssb/config`:

```json
{
  "decent": {
    "host": "127.0.0.1",
    "port": 8888
  }
}
```

If the Decent UI is behind a reverse proxy, set `wsRemote` to the public
WebSocket address:

```json
{
  "decent": {
    "wsRemote": "wss://your.domain:443"
  }
}
```

## Key modules

| Module | Purpose |
|---|---|
| `src/modules/core/sbot.js` | WebSocket connection to sbot; exposes pull-stream sources |
| `src/modules/core/app.js` | Top-level layout, navbar, hash routing |
| `src/modules/core/crypto.js` | Client-side message signing via `ssb-validate` |
| `src/modules/ui/names.js` | Resolve feed IDs to display names from `about` messages |
| `src/modules/ui/public.js` | Public timeline from followed feeds |
| `src/modules/ui/avatar-profile.js` | Profile view and editor |
| `src/modules/ui/compose.js` | Compose and publish posts |

See `AGENTS.md` in the repo root for development conventions and protocol notes.
